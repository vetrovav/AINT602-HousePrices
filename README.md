# AINT602 Week 2 Starter Repo: Git for ML Workflows

This repository is used in Week 2 of AINT602 to practise basic Git workflows in the context of machine learning.

The goal is not to learn Git from zero. The goal is to understand how Git supports reproducible ML workflows — and where Git is not enough.

## What this repo contains

- A simple Iris classification example
- A configuration file for model settings
- A training script
- A `.gitignore` file designed for ML projects
- Placeholder folders for data, models, and notebooks
